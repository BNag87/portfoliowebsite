Function Operation [Python]

This function takes an existing value from a list/array and generates a new number from it. If passed as a parameter to the dice roller above, it can be used to add a bonus to the results of a dice roll. This is an example of attribute modification in the world of DnD.

Code Block:


# this nested list is required to work with the function below
# the function will get the value stored in the 9th position

 charList = [["The Player", 33, 40, "Protagonist", 1, 2, False, 5, 3, 5],
 ["The Strongman", 40, 30, "Strongman", 1, 2, False, 5, 1, 3]]


# this function will set a bonus to speed rolls depending on the speed attribute
 def generate_player_speed_bonus():

  player_speed_bonus = 0
  if charList[0][9] == 1:
   player_speed_bonus = -3
   return player_speed_bonus

  elif charList[0][9] == 3:
   player_speed_bonus = 0
   return player_speed_bonus

  elif charList[0][9] == 5:
   player_speed_bonus = 3
   return player_speed_bonus

  else:
   print("Something went wrong when setting the player_speed_bonus. Setting it to 0")
   return player_speed_bonus
